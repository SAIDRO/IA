# Proyecto de Reconocimiento Facial con CNN

Este proyecto utiliza Redes Neuronales Convolucionales (CNN) para detectar y reconocer rostros en tiempo real.

## Instalación del Modelo
Debido a las políticas de tamaño de archivos de GitHub, el archivo del modelo entrenado no está incluido en este repositorio, por lo que decidimos incluirlo externamente.

1. **Descarga el modelo** desde este enlace: https://drive.google.com/file/d/1bl3Sc3M6fHKlUzvH8ctfHXASWN62Q8mQ/view?usp=sharing
2. Se deberá descomprimir o coloca el archivo `modelo_facial_resnet_turbo.h5` en la carpeta principal del proyecto para ejecutarse correctamente.

Nota: El entorno virtual (`.venv`) ha sido excluido del repositorio mediante `.gitignore` para mantener el proyecto ligero y compatible.
